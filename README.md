# 慢光阅读 Slowglow Reader

一个本地、无登录、不上传文字的温柔朗读阅读器。

Slowglow Reader is a gentle local read-aloud reader for people who want reading to feel safer, softer, and easier to return to.

## 为什么做

慢光阅读不是为了催人读得更快，也不是为了证明谁的专注力够好。

它想做的是一件更朴素的事：把阅读变得不那么吓人。你可以复制一段文字，让它放大显示、朗读出来，并高亮当前读到的位置。走神了也没关系，从高亮处回来就好。

核心信念是：

> 阅读能力不是被逼出来的，而是在足够安全、足够合适的支持里，慢慢恢复和生长出来的。

## 功能

- 粘贴或读取剪贴板文字
- 大字号阅读区
- 系统语音朗读
- 当前词高亮
- 播放中调整语速
- 保留原文段落和换行
- 专注模式
- 温柔模式：把当前句托出来，让已读内容和非当前句轻轻退后
- 可选 Bionic Reading 风格显示
- 可选信息流/逻辑信号标注
- 本地关键词和关键句候选，帮助做笔记起步

## 运行方式

### 直接打开

双击 `index.html`。

### Mac 推荐方式

双击：

```text
OpenSlowglowReader.command
```

如果 macOS 阻止打开，可以右键这个文件，选择“打开”，再确认一次。

## 使用流程

1. 在网页、PDF 或文档里选中文字。
2. 按 `Command C` 复制。
3. 回到慢光阅读。
4. 点 `读取剪贴板`，或直接粘贴到左侧文本框。
5. 点 `从头播放`。

状态不好、文字太压迫时，可以点 `温柔模式`。

## 隐私

当前版本是纯本地网页工具。

- 不需要登录
- 不需要 API key
- 不上传你的文字
- 不使用在线 AI

朗读使用浏览器和系统提供的语音能力。

## 适合谁

这个工具可能适合：

- 读长文容易过载的人
- ADHD / 注意力容易漂移的人
- 英语非母语读者
- 做笔记时容易抓不住重点的人
- 曾经因为阅读、学习或评价而产生压力的人

它不是医疗工具，也不能替代专业支持。它只是一个更温柔的阅读入口。
